# CRANE-X7_cigarette_dispenser_robot

## 概要
CRANE-X7とmediaPipeを連携させてシガレットをピッキングするROS2パッケージです。

- **画像認識**: mediaPipeを用いて、指の本数を検知してトリガーとする。
- **アームの制御**: 検知結果に基づいて、シガレットのピッキング動作を行う。
